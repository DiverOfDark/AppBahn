## Contributing

AppBahn is source-available under the Elastic License 2.0.

We welcome:
- 🐛 Bug reports — open a GitHub Issue with reproduction steps
- 💡 Feature requests — start a GitHub Discussion
- 📖 Documentation fixes — PRs to the docs/ directory

We do not currently accept code contributions to the platform,
operator, web console, or CLI. AppBahn is developed using an
internal LLM-first methodology with comprehensive private test
coverage.

If you've found a security vulnerability, please email
security@appbahn.eu instead of opening a public issue.
